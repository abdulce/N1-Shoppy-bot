# N1-Shoppy-bot
Discord bot for Shoppy.gg

## Installation
- `git clone https://github.com/shibuya-one/N1-Shoppy-bot`
- Fill in the *settings.json* file
- Start *main.py*
- `(prefix)help` - Have fun!

## Shoppy API Key
- Your Shoppy API key can be found at https://shoppy.gg/user/settings

### Used modules
- discord (& discord.ext)
- requests
- os
- json
- datetime
